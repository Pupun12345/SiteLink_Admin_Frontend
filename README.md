# SiteLink Admin Panel

Modern and professional admin panel for managing worker verifications.

## Features

- 🔐 Secure admin login
- 📊 Dashboard with real-time statistics
- 👷 Worker verification workflow
- 🎨 Modern UI with smooth animations
- 📱 Responsive design

## Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Default Admin Credentials

Use the credentials configured in your backend:
- Email: `admin@sitelink.in`
- Password: Your configured admin password

## Tech Stack

- React 19
- React Router DOM
- Framer Motion (animations)
- Lucide React (icons)
- Axios (API calls)
- Vite (build tool)

## Pages

- `/admin/login` - Admin login page
- `/admin/dashboard` - Dashboard with statistics
- `/admin/workers` - Worker verification page
